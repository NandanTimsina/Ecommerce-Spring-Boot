# Ecommerce Spring Boot Project

This is a backend e-commerce project built using Spring Boot.  
I made this while learning and practicing backend development concepts like REST APIs, authentication, and database handling.

## Tech Used

- Java
- Spring Boot
- Spring Data JPA
- Spring Security (JWT)
- H2 / MySQL
- Maven

## Features

- User authentication (JWT based)
- Add product to cart
- View cart
- Update product quantity in cart
- Remove product from cart
- Basic product & category structure

## Project Structure

- controller → handles API requests
- service → business logic
- repository → database queries
- model → entities
- util → helper classes (like getting logged-in user)

## How to Run

1. Clone the repo
2. Open in IntelliJ / any IDE
3. Run `EcommerceApplication.java`
4. Use Postman to test APIs
